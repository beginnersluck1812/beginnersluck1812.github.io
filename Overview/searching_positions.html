How Do We Search Through These Positions?

In the last portion of this tutorial we talked about how our AI is going to search through all the possible future moves and positions in the game.

In this section we are going to write the code that allows us to do this.

Writing the Code

We're going to create a function called 'searchThroughPositions' that takes a paramater called 'currentPosition' which represents the current position we are at in the search. 


                   void searchThroughPositions(OthelloPosition currentPosition) {

                   }


We want our function to do the following: Iterate through all the moves of the 'currentPosition' For each move, play it and get the resultant position Check if the new position is in a game over state If not re-call the function with the new position.

We can do this by: 

            void searchThroughPositions(currentPosition) {
                forEach(int move : currentPosition.findAvailableMoves()) {
                    OthelloPosition newPosition = currentPosition.getPositionAfterPlayingMove( move );

                    if ( !newPosition.isTerminal() ) { //game isnt over
                        searchThroughPositions( newPosition );
                    }

                }
            }
